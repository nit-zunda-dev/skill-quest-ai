# Task 5.3: Tailwind 移行後の手動確認チェックリスト

**要件 1.1**: 既存画面のスタイルが維持されていること。

## 手順

1. フロントエンドで開発サーバーを起動する: `pnpm --filter @skill-quest/frontend dev`
2. ブラウザでアプリを開き、以下の画面を順に確認する。

## 確認項目

- [ ] **未認証・ランディング**: ヒーロー・価値提案・CTA が意図どおり表示され、フォント・色（スレート・インディゴ）・レイアウトが崩れていない。
- [ ] **未認証・ログイン/サインアップ**: 「冒険を始める」クリック後、Chronicle タイトルとメール/パスワード入力フォームが表示され、既存と同様の見た目である。
- [ ] **認証後・Genesis INTRO**: イントロ画面のレイアウト・背景・ボタンが維持されている。
- [ ] **認証後・Genesis QUESTIONS**: 質問ステップのフォーム・ラベル・ボタンが維持されている。
- [ ] **認証後・Genesis LOADING / RESULT**: 読み込み表示・結果表示のスタイルが維持されている。
- [ ] **認証後・ダッシュボード**: ダッシュボードのレイアウト・チャート・既存 UI が維持されている。

## 結果

上記を確認したら、このファイルのチェックボックスにチェックを入れ、問題がなければ Task 5.3 を完了とする。
