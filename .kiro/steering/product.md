# Product Overview

RPG風のタスク管理アプリケーション。日々のタスクをクエストとして管理し、AIが生成する物語と共にユーザーの成長を楽しみながら習慣化を実現する。

## Core Capabilities

- **Genesis（世界生成）**: サインアップ時にAIがプロローグと初期クラスを生成し、ユーザー固有の世界観を構築
- **Quest Board（タスク管理）**: デイリータスク・習慣・To-Doの登録・編集・削除
- **Narrative Check-in**: タスク完了時にAIが物語セグメントと報酬（XP等）を生成
- **Character Status**: HP・MP・XP・Level の可視化
- **Grimoire（冒険ログ）**: 生成された物語履歴の閲覧
- **AI Partner Chat**: AIパートナーとの対話

## Target Use Cases

- タスクの習慣化をゲーミフィケーションで支援したいユーザー
- RPG・物語要素を楽しみながら進捗を可視化したいユーザー
- AIが生成するパーソナライズされた物語体験を求めるユーザー

## Value Proposition

タスク管理とRPG要素を融合し、AIが一人ひとりに合わせた物語を紡ぐことで、単なるTo-Do管理を超えた「冒険体験」を提供する。経験値・レベル・クラスといった成長指標が可視化され、継続の動機づけとなる。
