# Product Overview

Skill Quest AI は、タスク管理をゲームにするアプリケーションです。ユーザーはタスクをクエストとして登録・完了し、AIが生成する物語とキャラクター成長を通じて習慣化を促します。

## Core Capabilities

- **Genesis（世界生成）**: 登録時にAIがプロローグと初期クラスを生成し、オンボーディングを物語化
- **Quest Board（クエスト管理）**: デイリータスク・習慣・To-Doの登録・編集・完了
- **Narrative Check-in**: タスク完了時にAIが物語の1セグメントと報酬（XP等）を生成
- **Character Status**: XP・Level の可視化と成長体験
- **Grimoire（冒険ログ）**: 過去に生成された物語の履歴閲覧
- **AI Partner Chat**: AIパートナーとの対話とコンテキストに応じた応答

## Target Use Cases

- 日々のタスクをゲームのように続けたいユーザー
- 習慣化を物語と報酬で支えたいユーザー
- AIとの対話でモチベーションを維持したいユーザー

## Value Proposition

タスク管理とAIナラティブを組み合わせた独自体験。完了のたびに物語が進み、キャラクターが成長するため、習慣化が続けやすくなります。
