# Skill Quest AI アプリ紹介動画ガイド

アプリ紹介動画でユーザー数を伸ばすためのコンセプト・構成・施策案です。  

---

## 1. 動画の「一言メッセージ」

**「退屈なタスク管理アプリを、ゲーム感覚で。」**

- 30秒で伝えるなら: **目標を入れるとタスクがクエストになり、クリアするたびにAIが紡ぐ物語とアイテムがもらえる。もらったアイテムでAIパートナーやアンドロイドを喜ばせよう！**

---

## 2. 狙うべき「見せ場」（優先順）

| 順位 | 見せ場 | 理由 |
|------|--------|------|
| 1 | **クエストクリア直後**（アイテムが表示される瞬間） | コア体験のハイライト。「やった」が物語と報酬で一気に届く。 |
| 2 | **Genesis**（名前・目標入力 → クラスとプロローグが生成される流れ） | 「自分だけの物語が始まる」オンボーディング。初見のインパクトが大きい。 |
| 3 | **クエストボード**（タスクがクエスト／シナリオとして並び、AIパートナーがいる画面） | 普通のTo-Doとの違いが一目で伝わる。 |
| 4 | **グリモワール**（蓄積したストーリーログ・成長の履歴） | 「続けた分だけ物語が残る」価値の可視化。 |

---

## 3. 動画の種類と推奨構成

### 3.1 メイン紹介動画（60〜90秒）

**目的**: 初見の人に「何のアプリか・なぜ他と違うか」を一気に伝え、ダウンロード／登録へつなげる。

| 秒数 | 内容 |
|------|------|
| 0–10秒 | **フック**：「TODOアプリ、続かなかった人、手を挙べて」など課題に共感。または「タスク管理をゲーム感覚でできるアプリです」と一言で説明。 |
| 10–25秒 | **Genesis**：名前と目標を入力 → クラス・プロローグが生成される様子を短く。テキスト「自分だけの物語が始まる」など。 |
| 25–45秒 | **クエストボード＋クリア**：クエスト一覧 → 1つタップして完了 → **アイテムが表示される瞬間**をしっかり見せる（ここが一番の売り）。 |
| 45–60秒 | **パートナー＋グリモワール**：AIパートナーとの会話 or ウィジェットを一瞬 → グリモワールで「続けると物語が溜まる」を短く。 |
| 60–75秒 | **価値の言語化**：「退屈なタスク管理アプリを、ゲーム感覚で。」「続けた分だけ、自分のストーリーが残る。」 |
| 75–90秒 | **CTA**：アプリ名・「今すぐ始める」／ストアリンク・ハッシュタグ。 |

**トーン**: ノベルゲー／TRPGっぽさを出しつつ、押しすぎない。「習慣化アプリ」より「自分の物語を紡ぐアプリ」に寄せる。

---

### 3.2 ショート動画（15〜30秒・SNS用）

**目的**: X（Twitter）・Instagram Reels・TikTok・YouTube Shorts で流し、興味を持った人をメイン動画 or アプリへ誘導。

- **15秒版**
  - 課題の一言（「タスク管理アプリ、続かない」）→ 目標入力でクエスト化 → **クエストクリアで物語＋アイテムがもらえる瞬間**だけ見せる → 「Skill Quest AI」＋リンク。
- **30秒版**
  - 上記に「AIパートナーと一緒に進める」「グリモワールに履歴が残る」を1カットずつ追加。

**ポイント**: 最初の3秒で「タスクがクエスト」「物語とアイテムがもらえる」のどちらかが伝わるようにする。

---

### 3.3 体験実況・Before/After（60秒前後）

**目的**: 「三日坊主だった自分が、このアプリで続くようになった」ストーリーで共感を引く。

- Before: 「タスクアプリ、何個か試したけど続かなかった」
- 転換: 「このアプリはタスクがクエストで、クリアするたびに物語とアイテムがもらえる」
- After: クエストクリアの画面＋グリモワールの蓄積を見せて「続けてると、自分のストーリーが溜まってくる」
- CTA: アプリ or 紹介動画へのリンク

---

## 4. ターゲット別のメッセージ例

- **20〜30代・スキル習慣が続かない人**
  - 「資格・プログラミング・語学……やりたいのに一人だと続かない。このアプリは、タスクがクエストになって、クリアするたびに物語とアイテムがもらえる。AIパートナーと一緒に、自分のシナリオを進める感じ。」

---

## 5. 運用のポイント

| 項目 | 推奨 |
|------|------|
| **長さ** | メイン: 60〜90秒。SNS: 15〜30秒を複数パターン用意。 |
| **投稿先** | X, Instagram Reels, TikTok, YouTube（Short＋通常）,  Product Hunt / デモサイトのヒーロー動画。 |
| **CTA** | 「今すぐ始める」「ストアで見る」「紹介動画はプロフィールリンク」など、1本1明確に。 |
| **測定** | 再生数・完走率・クリック（リンク）・ストア遷移。ショートは「最初の3秒で離脱しないか」を重点的に。 |
| **改善** | 「クエストクリアの瞬間」を見せる割合や長さでA/Bテスト。フックの文言も試す。 |

---

## 6. チェックリスト（制作前）

- [ ] コア体験の「クエストクリア → 物語＋報酬」が必ず入っているか
- [ ] 「退屈なTODOを、自分だけの物語に」または同等の一言が入っているか
- [ ] Genesis（自分だけの物語が始まる）を短くでも見せているか
- [ ] AIパートナー or グリモワールのどちらか（できれば両方）が触れられているか
- [ ] 動画の最後にアプリ名＋明確なCTAがあるか
- [ ] 15秒版は「3秒で何のアプリか／何がすごいか」が分かるか

---

このガイドに沿って、まずは **60秒のメイン紹介動画** と **15秒・30秒のショート** を1本ずつ作ると、ユーザー獲得の検証を始めやすいです。
