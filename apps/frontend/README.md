# Chronicle - Skill Quest AI Frontend

AI 機能はバックエンド API（Cloudflare Workers AI）経由で提供されます。

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `pnpm install`
2. バックエンド API の URL を `.env.local` に設定（オプション）:
   `VITE_API_URL=http://localhost:8787`
3. Run the app:
   `pnpm dev`
